# C-sharp-assessment

1. Included in this solution is a file called input.txt, which contains a list of names (first and last).
    First, open and read the contents of the file.
    Then sort the contents of the file by last name, then by first name.  
    Finally write this sorted list to a new file.
    You must use at least one try/catch block and you must create at least one custom class.

2. This application shows total females, males and total number of students from an 
    XML document when the "Count" button is pressed.  The problem is the program
    throws an error when the "Count" button is pressed.  You will need to correct the 
    error and make sure all three totals are correct.
